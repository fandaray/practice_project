# Задание: Модель бойлера с OPC UA сервером

## Что нужно реализовать

### 1. boiler_model.py — Модель бойлера
- Бойлер с двумя входными потоками (горячая и холодная вода)
- Один выходной поток
- Управление через клапаны на входах и выходе
- Логика изменения уровня и температуры воды

### 2. opc_server.py — OPC UA сервер
- Предоставляет данные модели через протокол OPC UA
- Принимает команды управления клапанами
- Порт: 4840

### 3. server_main.py — Главный запуск
- Запускает модель и сервер вместе
- Обновляет состояние каждую секунду

### 4. opcua_client.py — Клиент
- Подключается к OPC UA серверу
- Читает текущие данные (уровень, температура, клапаны)

### 5. history_logger.py — Логгер истории
- Отдельный процесс
- Каждые 5 секунд читает данные с OPC UA сервера
- Записывает историю в SQLite базу данных

## Начальные условия
- Бойлер пустой (0% заполнения)
- Клапан горячей воды открыт на 50%
- Клапан холодной воды открыт на 50%
- Клапан выхода открыт на 100%
- Температура горячей воды — 90°C
- Температура холодной воды — 10°C

## Требования
- Язык: Python 3.11
- Библиотека OPC UA: opcua==0.98.13
- Все зависимости в requirements.txt
- Код должен запускаться через Dockerfile
- Код кладётся строго в папку src/
